Dashboard Global de Terremotos
Visualización interactiva de terremotos a nivel mundial usando datos reales, con mapas y gráficos dinámicos.

Tecnologías: 
Python · pandas · Plotly

Qué hace?: 
Carga y limpia datos sísmicos globales con pandas
Representa los terremotos en un mapa interactivo mundial con Plotly
Permite explorar la distribución por magnitud, profundidad y localización geográfica
